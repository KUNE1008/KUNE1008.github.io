# Dingkun Wei Homepage

This repository is adapted from [Jon Barron's homepage template](https://jonbarron.github.io/) and is built entirely with HTML and CSS. Feel free to use it freely.

This repository contains the static source code for Dingkun Wei's personal academic homepage.

## Local preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deployment

This site is intended for GitHub Pages as a user site:

- repository name: `kune1008.github.io`
- final URL: `https://kune1008.github.io/`


## Editing notes

- Main content is written directly in `index.html`.
- Update the profile image by replacing `assets/self.png`.
- Update experience logos by replacing the corresponding files in `assets/`.
